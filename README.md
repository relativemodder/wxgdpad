# wxgdpad

## Dependencies on Linux

- cmake
- gcc
- fmt
- wxwidgets

If you're on Debian-based systems you should install -dev versions of the above packages for building it.



## Building on Linux:

```bash
git clone https://github.com/relativemodder/wxgdpad
cd wxgdpad
cmake -Bbuild
cmake --build build
```


**Boom!**
